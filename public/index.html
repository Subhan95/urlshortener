Url Shortener

Given a valid url endpoint of the form http://www.some.domain returns an object containing the given url and its shortened version

eg:

  • /api/http://www.google.co.in

    • gives

    {"originalUrl":"http://www.google.co.in","shortUrl":"https://sheltered-tundra-95244.herokuapp.com0/0"}

Going to the shortened url will redirect you to the url given as the parameter

https://sheltered-tundra-95244.herokuapp.com/0
